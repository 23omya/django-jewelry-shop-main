# django-jewelry-shop-main
The project is a web-based e-commerce platform for online jewellery shopping.
Developed using Python Django framework for secure and scalable backend.
Customers can register, login, browse jewellery, and place orders online.
Admin can manage jewellery products, categories, customers, and orders.
The system includes cart, checkout, and order management features.
Ensures secure authentication and efficient database management.
